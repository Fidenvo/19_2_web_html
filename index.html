Kiểm tra tài nguyên CSS và JS

Hướng dẫn kiểm tra đường dẫn và lỗi bằng DevTools

CSS không hoạt động?

Kiểm tra lại thuộc tính href trong thẻ link có đúng đường dẫn không.

  • Ví dụ đúng: href="css/style.css"
  • File CSS phải nằm trong thư mục css/

JS không hoạt động?

Kiểm tra lại thuộc tính src trong thẻ script có đúng đường dẫn không.

  • Ví dụ đúng: src="js/script.js"
  • File JS phải nằm trong thư mục js/

Cách dùng DevTools

  • Bấm F12 để mở DevTools
  • Chọn tab Console — xem có lỗi đỏ không
  • Chọn tab Network — xem CSS và JS có load được không
  • File báo lỗi 404 nghĩa là sai đường dẫn
